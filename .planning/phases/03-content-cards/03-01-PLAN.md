# Plan: Content Cards & Breaking News Section

**Phase:** 03-content-cards
**Plan:** 01
**Estimated scope:** Medium (~20-30 minutes)

## Objective

Create reusable article card components for various homepage layouts including a breaking news section, compact article lists, and the EXCLUSIVE badge component. This builds on the hero grid components from Phase 2 to complete the homepage content display system.

## Execution Context

**Read these files before starting:**
- `frontend/components/HeroArticleCard.tsx` - Existing card component patterns
- `frontend/components/CategoryBadge.tsx` - Badge component pattern
- `frontend/app/globals.css` - Existing CSS utilities
- `frontend/app/page.tsx` - Homepage integration point

**Conventions to follow:**
- Components: PascalCase files in `frontend/components/`
- CSS: Use CSS variables (`--clemson-orange`, `--clemson-purple`, `--clemson-dark-purple`)
- Typography: `font-heading` for Apotek headlines, `font-body` for Basic Sans
- Layout: No border radius, small gaps (gap-1) for newspaper feel
- Images: Use Next.js `Image` component with proper sizes

## Context

**Prior decisions affecting this work:**
- Flexbox over CSS Grid for easier height control
- No border radius - clean edge-to-edge design
- Small gaps (gap-1, 4px) between cards for newspaper feel
- Orange (`--clemson-orange`) for badges and accents
- Dark purple gradient overlays on images

**Dependencies:**
- CategoryBadge component (exists)
- WordPress API client with getPosts (exists)
- Clemson color CSS variables (exists)

## Tasks

### Task 1: Create ExclusiveBadge Component
**File:** `frontend/components/ExclusiveBadge.tsx`

Create a badge component for EXCLUSIVE content markers:
- White text on dark purple background (`--clemson-dark-purple`)
- Uppercase, bold text
- Small pill shape with minimal padding
- Props: `className` for positioning overrides

### Task 2: Create ArticleCard Component
**File:** `frontend/components/ArticleCard.tsx`

Create a versatile article card with multiple variants:
- `variant: "horizontal" | "vertical" | "compact"`
- **Horizontal:** Thumbnail left, text right (for lists)
- **Vertical:** Image top, text below (for grids)
- **Compact:** Small thumbnail, title only (for sidebar)

Features:
- Accepts `WPPost` and category info
- Shows category badge, title, date
- Optional EXCLUSIVE badge support
- Hover effect: subtle scale on image
- Links to `/blog/[slug]`

### Task 3: Create BreakingNewsSection Component
**File:** `frontend/components/BreakingNewsSection.tsx`

Create the breaking news section for homepage:
- Section header: "BREAKING NEWS" with orange accent line
- 4-column grid of article cards (vertical variant)
- Each card: image, category badge, title, date
- Responsive: 2 columns on tablet, 1 on mobile
- Takes `posts: WPPost[]` and `categories: WPCategory[]` props

### Task 4: Create CompactArticleList Component
**File:** `frontend/components/CompactArticleList.tsx`

Create a compact list view for sidebars/secondary content:
- List of horizontal article cards
- Each item: small thumbnail (80x60), title, category

### Task 5: Create SocialCTABar Component
**File:** `frontend/components/SocialCTABar.tsx`
