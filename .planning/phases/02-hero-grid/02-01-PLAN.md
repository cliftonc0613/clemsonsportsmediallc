- Layout: Large featured article (50% width) + 2x2 smaller grid (50% width)
- Featured Article: Full-height image with dark purple gradient overlay, orange category badge (top-left), author + date (bottom), large bold headline (bottom, white)
- Grid Articles: Image with gradient overlay, category badge (top-left), author + date, bold headline (white text)
